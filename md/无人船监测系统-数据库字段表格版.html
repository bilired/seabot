无人船监测系统数据库字段汇总（论文表格版）

表头统一为：字段名 / 数据类型 / 约束条件 / 字段说明

1. UserProfile（用户扩展信息表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
user_idbigintFK->auth_user.id, OneToOne, NOT NULL关联用户
avatarvarchar(500)NULL, DEFAULT ''头像 URL
mobilevarchar(20)NULL, DEFAULT ''手机号
real_namevarchar(50)NULL, DEFAULT ''真实姓名
nick_namevarchar(50)NULL, DEFAULT ''昵称
gendervarchar(2)NULL, DEFAULT ''性别编码（1男/2女）
addressvarchar(255)NULL, DEFAULT ''地址
descriptiontextNULL, DEFAULT ''个人介绍

2. DroneDevice（无人船设备表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
owner_idbigintFK->auth_user.id, NULL设备所属用户
ship_typevarchar(50)NOT NULL船型
lengthintNOT NULL长度(cm)
modelvarchar(50)NOT NULL型号
weightfloatNOT NULL重量(kg)
functionsvarchar(255)NOT NULL功能模块
imagevarchar(500)NULL, DEFAULT ''设备图片 URL
stream_urlvarchar(500)NULL, DEFAULT ''视频拉流地址
statusvarchar(20)NOT NULL, DEFAULT 'offline', choices(online/offline)设备状态
max_speedfloatNOT NULL最高航速(节)
created_atdatetimeNOT NULL, auto_now_add创建时间
updated_atdatetimeNOT NULL, auto_now更新时间

3. ImageTransferRecord（图片传输记录表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
owner_idbigintFK->auth_user.id, NULL记录归属用户
ship_modelvarchar(50)NOT NULL, DEFAULT '未知型号'船型/设备型号
image_uidvarchar(64)NOT NULL, UNIQUE图片唯一标识
timestampdatetimeNOT NULL图片时间戳
image_formatvarchar(16)NOT NULL, DEFAULT 'jpg'图片格式
resolutionvarchar(32)NULL, DEFAULT ''分辨率（如1920x1080）
file_size_mbfloatNOT NULL, DEFAULT 0文件大小(MB)
image_urlvarchar(500)NOT NULL图片 URL
created_atdatetimeNOT NULL, auto_now_add记录创建时间

4. WaterQualityData（水质监测数据表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
ship_modelvarchar(50)NOT NULL船型号
temperaturefloatNOT NULL水温(°C)
phfloatNOT NULLpH值
chlorophyllfloatNOT NULL叶绿素(μg/L)
salinityfloatNOT NULL盐度
dissolved_oxygenfloatNOT NULL溶解氧(mg/L)
conductivityfloatNOT NULL电导率(μS/cm)
turbidityfloatNOT NULL浊度(NTU)
algaeintNOT NULL蓝绿藻(cells/mL)
warning_codevarchar(50)NOT NULL, DEFAULT '正常'警告码
collection_timedatetimeNOT NULL, auto_now_add采集时间
connection_statusvarchar(20)NOT NULL, DEFAULT '在线'连接状态

5. NutrientData（营养盐数据表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
ship_modelvarchar(50)NOT NULL船型号
phosphatefloatNOT NULL磷酸盐(mg/L)
phosphate_timedatetimeNOT NULL磷酸盐获取时间
ammoniafloatNOT NULL氨氮
ammonia_timedatetimeNOT NULL氨氮获取时间
nitratefloatNOT NULL硝酸盐
nitrate_timedatetimeNOT NULL硝酸盐获取时间
nitritefloatNOT NULL亚硝酸盐
nitrite_timedatetimeNOT NULL亚硝酸盐获取时间
error_code1varchar(10)NOT NULL, DEFAULT '00'错误码1
error_code2varchar(10)NOT NULL, DEFAULT '00'错误码2
instrument_statusvarchar(20)NOT NULL, DEFAULT '正常'仪器状态
collection_timedatetimeNOT NULL, auto_now_add采集时间
connection_statusvarchar(20)NOT NULL, DEFAULT '在线'连接状态

6. BoatTrackRecord（船体轨迹记录表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
ship_modelvarchar(50)NOT NULL, INDEX船型号
ship_portintNULL, INDEX来源端口
boat_timestampvarchar(20)NULL设备时间原始值
device_timedatetimeNULL, INDEX设备时间
statusvarchar(50)NULL状态
latitudefloatNOT NULL纬度
longitudefloatNOT NULL经度
speedfloatNULL航速(节)
directionfloatNULL航向(度)
battery_voltagefloatNULL电压(V)
recorded_atdatetimeNOT NULL, auto_now_add, INDEX入库时间

7. UserActivity（用户活动日志表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
user_idbigintFK->auth_user.id, NOT NULL关联用户
activity_typevarchar(50)NOT NULL, DEFAULT 'other', choices活动类型
descriptionvarchar(255)NOT NULL活动描述
created_atdatetimeNOT NULL, auto_now_add创建时间

8. DashboardStats（仪表板统计表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
user_idbigintFK->auth_user.id, OneToOne, NOT NULL关联用户
total_tasksintNOT NULL, DEFAULT 0总任务数
completed_tasksintNOT NULL, DEFAULT 0完成任务数
active_projectsintNOT NULL, DEFAULT 0活跃项目数
total_salesdecimal(10,2)NOT NULL, DEFAULT 0总销售额
new_users_countintNOT NULL, DEFAULT 0新增用户数
updated_atdatetimeNOT NULL, auto_now更新时间

9. MonthlySalesData（月度销售表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
monthvarchar(10)NOT NULL月份（如1月）
salesintNOT NULL销售金额

10. UserGrowthData（用户增长表）

字段名数据类型约束条件字段说明
idbigintPK, 自增主键
datedateNOT NULL日期
new_usersintNOT NULL新增用户数
active_usersintNOT NULL活跃用户数
