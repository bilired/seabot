# 如何使用真实数据

## 方案1：无人船通过API上传数据 ✅ 推荐

### 1.1 API端点

**上传水质数据**
- URL: `POST http://localhost:8000/api/upload/water-quality/`
- 无需认证

**上传营养盐数据**
- URL: `POST http://localhost:8000/api/upload/nutrient/`
- 无需认证

### 1.2 使用示例

#### Python示例（无人船端）

```python
import requests
import random

# 配置
API_URL = "http://localhost:8000/api/upload/water-quality/"
SHIP_MODEL = "DL-3026"

# 模拟传感器读取数据
sensor_data = {
    "shipModel": SHIP_MODEL,
    "temperature": round(20 + random.uniform(0, 5), 2),
    "ph": round(7 + random.uniform(-0.5, 0.5), 2),
    "chlorophyll": round(10 + random.uniform(0, 10), 2),
    "salinity": round(30 + random.uniform(0, 5), 2),
    "dissolvedOxygen": round(6 + random.uniform(0, 3), 2),
    "conductivity": round(400 + random.uniform(0, 100), 2),
    "turbidity": round(1 + random.uniform(0, 3), 2),
    "algae": int(500 + random.uniform(0, 1000)),
    "warningCode": "正常"
}

# 上传数据
response = requests.post(API_URL, json=sensor_data)
print(response.json())
```

### 1.3 定时上传脚本

创建一个定时任务，每5分钟上传一次数据：

```python
# upload_data_scheduled.py
import requests
import time
import random
from datetime import datetime

API_URL = "http://localhost:8000/api/upload/water-quality/"
SHIP_MODEL = "DL-3026"

def upload_data():
    """上传传感器数据"""
    data = {
        "shipModel": SHIP_MODEL,
        "temperature": round(20 + random.uniform(0, 5), 2),
        "ph": round(7 + random.uniform(-0.5, 0.5), 2),
        "chlorophyll": round(10 + random.uniform(0, 10), 2),
        "salinity": round(30 + random.uniform(0, 5), 2),
        "dissolvedOxygen": round(6 + random.uniform(0, 3), 2),
        "conductivity": round(400 + random.uniform(0, 100), 2),
        "turbidity": round(1 + random.uniform(0, 3), 2),
        "algae": int(500 + random.uniform(0, 1000)),
        "warningCode": "正常"
    }
    
    try:
        response = requests.post(API_URL, json=data)
        if response.status_code == 201:
            print(f"[{datetime.now()}] 数据上传成功")
        else:
            print(f"[{datetime.now()}] 上传失败: {response.text}")
    except Exception as e:
        print(f"[{datetime.now()}] 错误: {e}")

if __name__ == "__main__":
    print("开始定时上传数据...")
    while True:
        upload_data()
        time.sleep(300)  # 每5分钟上传一次
```

## 方案2：通过Django Admin手动管理数据

### 2.1 注册模型到Admin

编辑 `back-end/seadrone/accounts/admin.py`:

```python
from django.contrib import admin
from .models import WaterQualityData, NutrientData

@admin.register(WaterQualityData)
class WaterQualityDataAdmin(admin.ModelAdmin):
    list_display = ['ship_model', 'temperature', 'ph', 'chlorophyll', 
                    'warning_code', 'collection_time', 'connection_status']
    list_filter = ['ship_model', 'warning_code', 'connection_status']
    search_fields = ['ship_model']
    ordering = ['-collection_time']

@admin.register(NutrientData)
class NutrientDataAdmin(admin.ModelAdmin):
    list_display = ['ship_model', 'phosphate', 'ammonia', 'nitrate', 
                    'instrument_status', 'collection_time']
    list_filter = ['ship_model', 'instrument_status']
    search_fields = ['ship_model']
    ordering = ['-collection_time']
```

1. 创建超级用户（如果还没有）：
```bash
cd back-end/seadrone
python3 manage.py createsuperuser
```

2. 访问 `http://localhost:8000/admin/`

3. 手动添加、编辑、删除数据

## 方案3：从CSV/Excel导入数据

### 3.1 准备CSV文件

创建 `water_quality.csv`:
```csv
ship_model,temperature,ph,chlorophyll,salinity,dissolved_oxygen,conductivity,turbidity,algae,warning_code
DL-3026,22.5,7.2,15.3,32.1,8.2,450,2.5,850,正常
DL-3022,21.8,7.1,12.8,31.9,8.5,455,2.2,650,正常
```

### 3.2 导入脚本

```python
# import_csv_data.py
import os
import django
import csv

os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'seadrone.settings')
django.setup()

from accounts.models import WaterQualityData

def import_water_quality_csv(filename):
    """从CSV导入水质数据"""
    with open(filename, 'r', encoding='utf-8') as file:
        reader = csv.DictReader(file)
        count = 0
        for row in reader:
            WaterQualityData.objects.create(
                ship_model=row['ship_model'],
                temperature=float(row['temperature']),
                ph=float(row['ph']),
                chlorophyll=float(row['chlorophyll']),
                salinity=float(row['salinity']),
                dissolved_oxygen=float(row['dissolved_oxygen']),
                conductivity=float(row['conductivity']),
                turbidity=float(row['turbidity']),
                algae=int(row['algae']),
                warning_code=row['warning_code'],
                connection_status='在线'
            )
            count += 1
        print(f"成功导入 {count} 条数据")

if __name__ == "__main__":
    import_water_quality_csv('water_quality.csv')
```

## 方案4：对接第三方水质监测系统

如果有现有的水质监测系统API，可以定期拉取数据：

```python
# sync_from_external_api.py
import requests
import os
import django

os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'seadrone.settings')
django.setup()

from accounts.models import WaterQualityData

def sync_from_external_system():
    """从外部系统同步数据"""
    # 假设外部系统API
    external_api = "https://external-system.com/api/water-data"
    
    response = requests.get(external_api)
    if response.status_code == 200:
        data = response.json()
        
        for item in data:
            WaterQualityData.objects.create(
                ship_model=item['device_id'],
                temperature=item['temp'],
                ph=item['ph_value'],
                # ... 映射其他字段
            )
        print(f"同步了 {len(data)} 条数据")

if __name__ == "__main__":
    sync_from_external_system()
```

## 推荐方案总结

| 方案 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| **API上传** | 无人船实时上传 | 实时、自动化 | 需要设备联网 |
| **Admin管理** | 少量数据手动录入 | 简单直观 | 不适合大量数据 |
| **CSV导入** | 批量历史数据导入 | 一次性导入大量数据 | 不是实时的 |
| **第三方对接** | 已有监测系统 | 利用现有数据 | 需要API对接 |

**建议**：对于无人船项目，使用**方案1（API上传）**最合适，设备每隔一定时间（如5分钟）自动上传传感器数据到服务器。
