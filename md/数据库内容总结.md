## 🔄 API 接口概览

- `POST /api/login/` - 用户登录
- `GET /api/user/info/` - 获取用户信息

### 菜单和权限
- `GET /api/v3/system/menus/` - 获取菜单列表

### 仪表板
- `GET /api/dashboard/stats/` - 获取统计数据
- `GET /api/dashboard/activity/` - 获取用户活动
- `GET /api/dashboard/sales/` - 获取销售数据
- `GET /api/dashboard/growth/` - 获取用户增长数据

### 设备管理
- `GET /api/drone/list/` - 获取设备列表
- `POST /api/drone/create/` - 创建设备
- `POST /api/drone/delete/` - 删除设备
- `POST /api/drone/batch-delete/` - 批量删除设备

### 水质监测
- `GET /api/analysis/water-quality/` - 获取水质数据
- `GET /api/analysis/nutrient/` - 获取营养盐数据
- `POST /api/upload/water-quality/` - 上传水质数据
