# 添加设备功能说明

## 功能概述

在无人船设备管理页面中，已成功实现了"添加设备"功能。用户可以通过该功能向系统添加新的无人船设备。

## 功能位置

**页面路径**: `/system/drone` (无人船设备管理页面)

**按钮位置**: 表格头部左侧的"添加"按钮

## 使用流程

### 1. 打开添加设备对话框
- 点击表格头部的"添加"按钮
- 系统会弹出一个"添加新设备"对话框

### 2. 填写设备信息

对话框中包含以下必填字段：

| 字段名 | 说明 | 示例 |
|--------|------|------|
| **船型** | 设备的船体类型 | 双体、单体 |
| **型号** | 设备的型号编号 | DL-3022、DL-3026 |
| **长度(m)** | 设备的长度，单位米 | 220、255 |
| **重量(kg)** | 设备的重量，单位公斤 | 44、51 |
| **最高航速(节)** | 设备的最高航速，单位节 | 12、14 |
| **功能模块** | 设备具备的功能 | 图传、采样、水质监测 |

### 3. 提交表单
- 填写完所有必填字段后，点击"提交"按钮
- 系统会校验表单数据
- 如果验证通过，会向后端 API 提交创建请求

### 4. 结果反馈
- **成功**: 显示"设备添加成功"提示，对话框自动关闭，设备列表自动刷新
- **失败**: 显示错误信息提示，用户可以修正后重新提交

## 表单验证规则

所有字段都是必填项，验证规则如下：

```javascript
{
  shipType: [{ required: true, message: '请输入船型' }],
  model: [{ required: true, message: '请输入型号' }],
  length: [{ required: true, message: '请输入长度' }],
  weight: [{ required: true, message: '请输入重量' }],
  maxSpeed: [{ required: true, message: '请输入最高航速' }],
  functions: [{ required: true, message: '请输入功能模块' }]
}
```

### 请求地址
```
POST /api/drone/create/
```

### 请求体
```json

  "maxSpeed": 12,
```

#### resetForm() - 重置表单
```typescript
const resetForm = () => {
  formRef.value?.resetFields()
  formData.value = {
    shipType: '',
    model: '',
    length: 0,
    weight: 0,
    maxSpeed: 0,
    functions: ''
  }
}
```

#### submitAddDevice() - 提交表单
```typescript
const submitAddDevice = async () => {
  if (!formRef.value) return
  try {
    await formRef.value.validate()  // 表单验证
    submitting.value = true
    
    // 调用后端 API
    const response = (await createDrone({...})) as any
    
    if (response?.data?.code === 200) {
      ElMessage.success('设备添加成功')
      addDialogVisible.value = false
      await refreshData()  // 刷新列表
    } else {
      ElMessage.error(response?.data?.msg || '添加失败')
    }
  } catch (e) {
    console.error('添加设备失败:', e)
    ElMessage.error('添加设备失败，请重试')
  } finally {
    submitting.value = false
  }
}
```

## 交互细节

### 对话框关闭行为
- 点击"取消"按钮：关闭对话框（不保存数据）
- 点击"提交"按钮：验证表单并提交
- 点击对话框右上角关闭按钮：关闭对话框，触发 `@close="resetForm"`，自动清空表单数据

### 加载状态
- 提交时，"提交"按钮会显示加载状态 (`:loading="submitting"`)
- 防止用户在等待期间多次点击

### 表单校验反馈
- 验证失败时，输入框会显示红色边框和错误提示
- 用户需要修正后才能继续提交

## 错误处理

### 可能的错误情况

1. **表单验证失败**
   - 用户未填写必填字段
   - Element Plus 自动显示验证错误提示

2. **网络请求失败**
   - 后端服务不可用
   - 显示"添加设备失败，请重试"

3. **后端业务错误**
   - 例如：设备型号已存在、参数非法等
   - 显示后端返回的错误信息

4. **认证失败**
   - 用户未登录或 Token 过期
   - Axios 拦截器会自动处理，重定向到登录页

## 注意事项

- **必填字段**: 所有字段都是必填的，不能为空
- **数字字段**: 长度、重量、最高航速必须是有效的数字
- **状态**: 新添加的设备初始状态为"离线"
- **权限**: 需要登录后才能使用此功能
- **刷新**: 添加成功后，设备列表会自动刷新，显示新添加的设备
