# 轨迹查询功能配置说明

## Mapbox Token 配置

轨迹查询功能使用 Mapbox GL JS 来显示地图和轨迹。你需要配置一个有效的 Mapbox Access Token。

### 获取 Mapbox Access Token

1. 访问 [Mapbox 官网](https://www.mapbox.com/)
2. 注册/登录账号
3. 进入 [Access Tokens 页面](https://account.mapbox.com/access-tokens/)
4. 复制你的 **Default public token** 或创建一个新的 token

### 配置 Token

在文件 `src/views/system/drone/modules/track-dialog.vue` 中，找到第 56 行：

```typescript
const MAPBOX_TOKEN = 'pk.eyJ1IjoieW91cnVzZXJuYW1lIiwiYSI6InlvdXJ0b2tlbiJ9.example'
```

将其替换为你的实际 token：

```typescript
const MAPBOX_TOKEN = 'pk.eyJ1Ijoi...' // 粘贴你的 token
```

### 功能说明

轨迹查询对话框包含：

- **左侧时间列表**: 显示历史轨迹记录，点击可切换查看不同时间段的轨迹
- **右侧地图**: 使用 Mapbox 显示轨迹路径
  - 绿色标记：起点
  - 红色标记：终点
  - 蓝色线条：轨迹路径

### 自定义地图样式

在 `track-dialog.vue` 第 108 行，可以修改地图样式：

```typescript
style: 'mapbox://styles/mapbox/streets-v12', // 街道地图
```

可选样式：
- `mapbox://styles/mapbox/streets-v12` - 街道地图
- `mapbox://styles/mapbox/satellite-v9` - 卫星图
- `mapbox://styles/mapbox/outdoors-v12` - 户外地图
- `mapbox://styles/mapbox/light-v11` - 浅色地图
- `mapbox://styles/mapbox/dark-v11` - 深色地图

### 使用方法

1. 在无人船设备管理页面，点击任意设备的"轨迹查询"按钮
2. 弹出轨迹查询对话框
3. 左侧选择要查看的历史记录
4. 右侧地图会显示相应的轨迹路径

### 数据集成

目前使用模拟数据，要集成真实数据，请在 `track-dialog.vue` 中修改：

1. 将 `trackHistory` 改为从 API 获取
2. 确保数据格式为：
   ```typescript
   {
     time: '2024-08-02 15:00',
     coordinates: [[lng1, lat1], [lng2, lat2], ...]
   }
   ```
   其中 coordinates 是 [经度, 纬度] 数组

## 注意事项

- Mapbox 免费套餐每月有 50,000 次地图加载限额
- 坐标格式为 [经度, 纬度]，注意顺序
- 如果地图无法加载，请检查：
  1. Token 是否有效
  2. 网络连接是否正常
  3. 浏览器控制台是否有错误信息
