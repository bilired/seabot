## 🚀 快速开始（3个核心步骤）

### 步骤 1️⃣：理解部署流程（5分钟）

**打开并阅读：** `部署总结.md`
- 了解整体流程
- 检查 Python 版本问题（⚠️ 重要）
- 预估时间 45-60 分钟

**关键点：**
- ⚠️ 你的服务器 Python 3.7.9 需要升级到 3.11+
- 宝塔面板中通过"软件商店"安装 Python 3.11
- 部署时需要 SECRET_KEY 和数据库密码

---

### 步骤 2️⃣：宝塔面板准备（15分钟）

**在宝塔面板中完成：**

1. ✅ **安装 Python 3.11**
   - 宝塔 → 软件商店 → 搜索 Python → 选择 3.11 安装

2. ✅ **添加网站**
   - 宝塔 → 网站 → 添加网站
   - 域名：`yunpingtai.cc` + `www.yunpingtai.cc`

3. ✅ **申请 SSL 证书**
   - 点击网站 → SSL → Let's Encrypt
   - 勾选"自动续期"

4. ✅ **创建数据库**
   - 宝塔 → 数据库 → 添加数据库
   - 名称：`seaboat_prod`
   - 用户：`seaboat_user`
   - 密码：设置强密码并 **保存** ⚠️

---

### 步骤 3️⃣：执行部署脚本（30分钟）

**打开文档：** `部署速查卡.md`（搜索"SSH 登录"部分）

**简单命令流：**
```bash
# SSH 登录
ssh root@yunpingtai.cc

# 进入项目目录
cd /home/wwwroot/yunpingtai.cc

# 1. 生成密钥（复制完整输出）
python3 -c "import secrets; print(secrets.token_urlsafe(50))"

# 2. 编辑环境变量
cp .env.example .env
nano .env
# 修改：SECRET_KEY 和 DB_PASSWORD

# 3. 自动化部署
chmod +x deploy.sh
sudo bash deploy.sh

# 4. 启动服务
sudo cp seaboat-gunicorn.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable seaboat-gunicorn
sudo systemctl start seaboat-gunicorn

# 5. 配置 Nginx
# 方式A：宝塔 UI → 网站 → 配置文件 → 粘贴 nginx.conf
# 方式B：sudo cp nginx.conf /usr/local/nginx/conf/vhost/yunpingtai.cc.conf
```
