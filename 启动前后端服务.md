# 启动前后端服务步骤

## 1️⃣ 启动后端 Django 服务 (必须先启动)

### 打开第一个终端窗口：

```bash
# 进入后端目录
cd /Users/hanksgao/Desktop/art-design-menu/back-end/seadrone

# 激活虚拟环境（如果还没激活）
source ../myenv/bin/activate

# 迁移数据库（如果还没执行过）
python manage.py migrate

# 创建超级用户（如果还没创建过）
# python manage.py createsuperuser

# 启动 Django 开发服务器（监听 8000 端口）
python manage.py runserver 0.0.0.0:8000
```

**成功输出示例：**
```
Starting development server at http://127.0.0.1:8000/
Quit the server with CONTROL-C.
```

✅ **确认后端已启动**：访问 http://localhost:8000/admin

## 📋 创建测试用户（如果登录失败）

如果显示 "用户名或密码错误"，需要在 Django 中创建用户。

在第一个终端（Django 服务器的终端）中，**按 CTRL+C 停止服务器**，然后：

```bash
# 进入 Django shell
python manage.py shell

# 执行以下 Python 代码：
from django.contrib.auth.models import User
User.objects.create_user('Admin', password='123456')
User.objects.create_user('Super', password='123456')
User.objects.create_user('User', password='123456')
exit()
```

然后重新启动 Django 服务器：
```bash
python manage.py runserver 0.0.0.0:8000
```

## 🔍 调试技巧

### 查看前端网络请求
1. 打开浏览器开发者工具 (F12)
2. 切换到 "Network" 标签
3. 尝试登录，观察请求：
   - 请求 URL 应该是：`http://localhost:3006/api/login/`
   - 请求方法：`POST`
   - 响应状态：应该是 `200` (成功) 或 `401` (密码错误)

### 查看后端日志
Django 控制台会显示每个请求的日志，例如：
```
[23/Jan/2026 10:30:45] "POST /api/login/ HTTP/1.1" 200 ...
```

### 常见问题排查

| 错误 | 原因 | 解决 |
|------|------|------|
| 404 Not Found | 后端路由不对 | 确保 Django 已启动，URL 配置正确 |
| Connection refused | 后端没运行 | 启动 Django 服务器 |
| CORS 错误 | 跨域问题 | 确保 Django 已安装 corsheaders 配置 |
| 用户名或密码错误 | 用户不存在 | 创建测试用户 |

## 📝 架构流程

```
┌─────────────────────┐
│  浏览器 (localhost:3006)
└──────────┬──────────┘
           │ 请求 /api/login/
           ▼
┌─────────────────────────────────────┐
│  Vite 开发服务器 + 代理             │
│  配置代理：/api/ → localhost:8000    │
└──────────┬──────────────────────────┘
           │ 转发请求
           ▼
┌─────────────────────────────────────┐
│  Django 服务器 (localhost:8000)     │
│  接收 POST /api/login/              │
│  返回 JSON: { token, refreshToken } │
└──────────┬──────────────────────────┘
           │ 返回响应
           ▼
┌─────────────────────────────────────┐
│  Vue 前端                           │
│  存储 token，跳转到首页             │
└─────────────────────────────────────┘
```

## ⚡ 快速启动脚本

**创建文件：`start-dev.sh`**

```bash
#!/bin/bash

# 启动 Django
echo "🚀 启动 Django 服务器 (端口 8000)..."
cd back-end/seadrone
source ../myenv/bin/activate
python manage.py runserver 0.0.0.0:8000 &
DJANGO_PID=$!

sleep 2

# 启动 Vue
echo "🚀 启动 Vue 开发服务器 (端口 3006)..."
cd ../../../art-design-pro
pnpm dev &
VUE_PID=$!

echo "✅ 两个服务都已启动！"
echo "📍 前端: http://localhost:3006"
echo "📍 后端: http://localhost:8000"

# 保持脚本运行
wait
```

**使用：**
```bash
chmod +x start-dev.sh
./start-dev.sh
```
